# Redesign of Dicasonline
Redesign of the homepage of Dicasonline (https://www.dicasonline.com/), with focus on mobile-first design and good UX. 

### Run Project

Step 1.

```
git clone redesign-dicasonline repo
```

Step 2.

```
cd redesign-dicasonline
```

Step 3.

```
npm install
```

Step 4.

```
npm run dev
```
